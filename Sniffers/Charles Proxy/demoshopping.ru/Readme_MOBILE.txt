Задача для смартфона

1 Все действия со сниффером и браузером надо записать на видео.
2 Добавьте несколько разных товаров в корзину, отправьте запрос на удаление любого из них из корзины, но видоизмените его так, чтобы он удалил другой товар, на уже известном ресурсе https://intern.demoshopping.ru
3 Смоделируйте ситуацию, при которой при обращении к https://intern.demoshopping.ru пользователь увидит в браузере любую картинку.
4 Приложите скриншот любого перехваченного HTTPs-запроса с вашего мобильного устройства. В header user-agent должна быть информация о вашем устройстве.
